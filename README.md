Pizza Sales Analysis & Performance Dashboard
This project explores a pizza restaurant's sales data to uncover business performance trends, customer preferences, and revenue drivers. The project is an end-to-end data analysis solution using SQL for data processing and Power BI for interactive visualization.


Project Objectives
The analysis focuses on key business metrics to drive strategic growth:
Sales Trends: Identifying peak order times on a daily and monthly basis.
Product Performance: Analyzing the best and worst-performing pizzas by revenue and order volume.
Category Insights: Understanding the distribution of sales across different pizza categories and sizes.


Tech Stack
SQL Server: Data cleaning, KPI calculation, and complex query development.
Power BI Desktop: Data modeling, DAX measures, and interactive dashboard design.
Excel/CSV: Raw data management.


Key Business Insights;
-Revenue & Order Highlights
Top Revenue Generator: The Thai Chicken Pizza ($43,434)
Most Ordered Pizza: The Classic Deluxe Pizza (2,329 Orders)
Lowest Performing Pizza: The Brie Carre Pizza ($11,588 revenue / 480 orders)

-Sales Trends
Weekly Patterns: Order volume peaks significantly on weekends, particularly Friday and Saturday evenings.
Category Dominance: The Classic category leads in total order quantity.
Size Preference: Large (L) size pizzas contribute the most to total revenue.
